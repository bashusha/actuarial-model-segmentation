# When Is Segmentation Needed in Actuarial Models?

This repository contains the LaTeX sources and PDF of the paper

**"When Is Segmentation Needed in Actuarial Models?  
Diagnosing Structural Incompatibility"**

by Xana Verte (2026).

This work is prepared for submission to the **7th African Actuarial Congress (7AAC)** and is aligned with the conference’s Call for Papers.


The paper proposes a diagnostic framework for assessing whether portfolio
segmentation is structurally justified in actuarial models, particularly
under conditions of limited data and strong regulatory constraints.
